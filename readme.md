# Description

Dawid Majewski's Frontend Portfolio.

## Used technologies

- WTF Starter Kit
- HTML/CSS/JS
- GitHub API

## Available scripts

`npm run start` - runs development mode

`npm run build` - runs build process for production

`npm run publish` - runs build process and publish the page using `gh-pages` branch
